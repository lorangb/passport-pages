# Passport Pages

**Travel books that make every destination come alive for kids.**

A business that creates beautiful, age-appropriate travel activity/fact books for children aged 5–16+.
Every major city and destination worldwide, available in 12+ languages, in digital PDF or premium hardback formats.

---

## The Concept

Parents traveling with kids want their children to be engaged, curious, and excited about the places they visit. But most travel guides are written for adults — dry, text-heavy, no fun.

**Passport Pages** fills that gap with destination-specific books tailored to three age groups:

| Edition | Ages | Pages | Focus |
|---------|------|-------|-------|
| Young Explorer | 5–8 | 64 | Big illustrations, coloring, seek-and-find, simple facts |
| Middle Explorer | 9–12 | 148 | Nat Geo–style facts, world records, puzzles, philosophy |
| Trailblazer | 13–16+ | 200+ | Deep history, cultural insights, essay-style, photography |

## Destination Tiers

- **Tier 1:** Global icons (London, Paris, Tokyo, Rome, NYC...) — $24.95 PDF / $49.95 hardback
- **Tier 2:** Regional treasures (Brighton, Oxford, Bruges, Edinburgh...) — $19.95 PDF / $39.95 hardback
- **Tier 3:** Hidden gems (Cotswolds, Hallstatt, Cinque Terre...) — $14.95 PDF / $29.95 hardback

## Languages

All books available simultaneously in: English, Spanish, French, German, Italian, Portuguese, Mandarin, Japanese, Korean, Arabic, Dutch, Russian, and more.

## Getting Started

This project is designed to be built with OpenCode. The storefront website is fully designed and ready to go. Next steps:

2. Design and publish the first book (London — Middle Explorer)
3. Set up Stripe for payments
4. Set up Print-on-Demand for hardbacks
5. Build out all 200+ destinations
